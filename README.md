# Alien Invesion Game 

Alien Invasion is a beginner‑friendly Python game based on the project from Python Crash Course. In this game, we control a spaceship that moves across the bottom of the screen and shoots bullets upward to stop waves of incoming aliens. The goal is to clear each wave before the aliens reach the ship or the bottom of the screen.

This project helps students practice core programming skills like working with classes, handling user input, updating game objects, detecting collisions, and managing game loops using the Pygame library. I will continue developing and improving the game and this file as I learn more about the game.
